# Python图书借阅管理系统 python254

本项目是一款基于Python语言的图书借阅管理系统，集成了用户管理、图书信息管理、借阅与归还功能，并提供了丰富的统计功能，帮助管理员高效管理图书借阅流程。

## 技术栈

- **后端**：Python、Flask框架
- **前端**：HTML、CSS、JavaScript、jQuery
- **数据库**：MySQL 8.0
- **开发工具**：PyCharm、Navicat

## 系统角色

- **普通用户**：可进行登录注册、图书借阅、归还图书等操作。
- **管理员**：负责图书信息管理、用户管理、借阅统计、图书统计等。

## 运行环境

- Python 3.0及以上版本
- MySQL 8.0

## 功能模块

- **登录注册**：用户可以通过注册账号，登录系统。
- **图书借阅**：用户可以查询、借阅图书。
- **归还图书**：用户可以在借阅期限内向系统归还图书。
- **图书信息管理**：管理员可以增删改查图书信息。
- **用户管理**：管理员对用户信息进行管理。
- **借阅统计**：统计用户的借阅信息。
- **图书统计**：统计图书的借阅情况。

1. 环境搭建：安装Python 3.0以上版本，配置MySQL 8.0数据库。
2. 克隆项目代码到本地。
3. 安装依赖：使用pip安装`requirements.txt`文件中列出的依赖。
4. 配置数据库：根据项目需求初始化数据库。
5. 运行项目：使用命令行启动Flask应用。
